## 📊 一、系统概览

### 1.1 核心功能
- **NPC 对话生成**：所有 NPC（固定 + 随机）的实时对话
- **人设管理**：NPC 角色定义、性格、背景故事
- **语言规则**：强制英语对话、动作词格式、双语显示
- **场景设定**：不同场景的对话约束（机场、卢奥晚宴等）
- **Prompt 管理**：预定义的对话提示词库

### 1.2 性能瓶颈
- ❌ **重复代码**：多个 `getAIResponse` 包装函数
- ❌ **冗余逻辑**：人设获取分散在多处
- ❌ **Prompt 未充分利用**：场景设定未自动应用
- ❌ **调试日志过多**：影响性能（已部分清理）

### 2.2 核心函数详解

#### 📍 `GameCore.api.getAIResponse(prompt, npcName, conversationHistory)`
**位置**: `core.js:185-240`  
**作用**: AI 对话生成核心入口

```javascript
async getAIResponse(prompt, npcName, conversationHistory) {
    // ========== 步骤 1: 定义语言规则 ==========
    const NPC_LANGUAGE_RULES = `【强制语言规则】
    ❗️❗️❗️ 你必须只说英语！绝对不要说中文！❗️❗️❗️
    1. 只说英语，绝对不要说中文
    2. 动作描述词必须用英文，并用斜体表示（如 *smiles*, *nods*, *laughs*）
    3. 绝对不要用中文动作词！不要用括号（）包含动作！
    4. 保持自然、真实的英语对话风格
    
    【对话格式】
    - 你只需要生成英语对话
    - 系统会自动在下方显示中文翻译
    - 上方 = 英语原文（带英文动作词）
    - 下方 = 中文翻译（由系统生成）
    - 你不需要自己写中文翻译！`;
    
    // ========== 步骤 2: 获取 NPC 人设 ==========
    let npcRole = "你是一个友好的人，用英语交流，热情帮助他人。\n\n" + NPC_LANGUAGE_RULES;
    
    if (window.FIXED_NPCS && window.FIXED_NPCS[npcName]) {
        const npc = window.FIXED_NPCS[npcName];
        if (npc.rolePrompt) {
            npcRole = npc.rolePrompt;  // 使用预设 rolePrompt
        } else {
            // 生成标准 prompt + 语言规则
            const description = npc.description || npc.background || '';
            npcRole = NPC_LANGUAGE_RULES + '\n\n你是' + npc.name + '，' + 
                     npc.age + '岁，' + npc.nationality + '，职业是' + 
                     npc.occupation + '。' + npc.personality + '。' + description;
        }
    }
    
    // ========== 步骤 3: 构建 AI 请求 ==========
    const messages = [
        { role: "system", content: npcRole },  // NPC 人设 + 语言规则
        ...conversationHistory,                 // 对话历史
        { role: "user", content: prompt }      // 用户输入
    ];
    
    // ========== 步骤 4: 调用 DeepSeek API ==========
    const response = await fetch(GameCore.config.apiUrl, {
        method: "POST",
        headers: {
            "Content-Type": "application/json",
            "Authorization": `Bearer ${GameCore.methods.getCurrentAPIKey()}`
        },
        body: JSON.stringify({
            model: "deepseek-chat",
            messages: messages,
            max_tokens: 800,
            temperature: GameCore.config.defaultTemperature
        })
    });
    
    const data = await response.json();
    return data.choices[0].message.content;
}
```

**关键点**:
- ✅ 内联语言规则，确保所有 NPC 遵守
- ✅ 支持 FIXED_NPCS、AIRPORT_NPCS、LUAU_NPCS
- ✅ 构建标准的 OpenAI 格式 messages

#### 📍 `getNpcRole(npcName)`
**位置**: `npc.js:52-86`  
**作用**: 获取 NPC 角色定义

```javascript
function getNpcRole(npcName) {
    let npc = null;
    
    // 优先级：FIXED_NPCS > AIRPORT_NPCS > LUAU_NPCS
    if (window.FIXED_NPCS && window.FIXED_NPCS[npcName]) {
        npc = window.FIXED_NPCS[npcName];
    } else if (window.AIRPORT_NPCS) {
        npc = window.AIRPORT_NPCS.find(n => n.name === npcName);
    } else if (window.LUAU_NPCS) {
        npc = window.LUAU_NPCS.find(n => n.name === npcName);
    }
    
    if (npc) {
        if (npc.rolePrompt) {
            // 如果 rolePrompt 已包含语言规则，直接返回
            if (npc.rolePrompt.includes('【语言规则】') || 
                npc.rolePrompt.includes('动作描述词必须用英文') || 
                npc.rolePrompt.includes('强制语言规则')) {
                return npc.rolePrompt;
            }
            // 否则添加语言规则
            return NPC_LANGUAGE_RULES + '\n\n' + npc.rolePrompt;
        }
        // 生成标准 prompt
        const description = npc.description || npc.background || '';
        return NPC_LANGUAGE_RULES + '\n\n你是' + npc.name + '，' + npc.age + '岁，' + 
               npc.nationality + '，职业是' + npc.occupation + '。' + 
               npc.personality + '。' + description;
    }
    
    // 默认角色
    return "你是一个友好的人，用英语交流，热情帮助他人。\n\n" + NPC_LANGUAGE_RULES;
}
```

**优先级**:
1. FIXED_NPCS（固定 NPC：海关官员、前台接待员、Maya 等）
2. AIRPORT_NPCS（机场邂逅 NPC：11 个随机角色）
3. LUAU_NPCS（卢奥晚宴 NPC：5 个神秘男人）
4. 默认角色

### 3.2 架构问题

#### ❌ 问题 1: 重复的 `getAIResponse` 函数

```javascript
// 1. core.js:185 - 核心实现
GameCore.api.getAIResponse(prompt, npcName, history)

// 2. aigenerate/ai.js:16 - 包装函数
async function getAIResponse(prompt, npcName, history) {
    return GameCore.api.getAIResponse(prompt, npcName, history);
}

// 3. core.js:329 - 全局快捷方式
window.getAIResponse = (p, n, h) => GameCore.api.getAIResponse(p, n, h);
```

**影响**:
- 调用链路不清晰
- 难以追踪问题
- 增加维护成本

**解决方案**: 删除包装函数，统一使用 `GameCore.api.getAIResponse`

---

#### ❌ 问题 2: 场景 Prompt 未自动应用

**当前流程**:
```
getAIResponse(npcName, prompt)
    ↓
获取 NPC 人设 (getNpcRole)
    ↓
添加语言规则 (NPC_LANGUAGE_RULES)
    ↓
调用 AI
```

**缺失**: 场景设定（如 `AIRPORT_CONVERSATION`、`LUAU_CONVERSATION`）

**影响**:
- 机场邂逅 NPC 不知道自己在机场
- 卢奥晚宴 NPC 不知道场景氛围
- 对话缺乏场景感

**解决方案**: 在 `getAIResponse` 中根据 NPC 类型自动添加场景 prompt

---

#### ❌ 问题 3: 人设获取逻辑分散

**当前代码**:
```javascript
// npc.js: getNpcRole 函数
if (window.FIXED_NPCS && window.FIXED_NPCS[npcName]) { ... }
else if (window.AIRPORT_NPCS) { ... }
else if (window.LUAU_NPCS) { ... }

// core.js: getAIResponse 内联实现
if (window.FIXED_NPCS && window.FIXED_NPCS[npcName]) { ... }
```

**影响**:
- 逻辑重复
- 修改一处需要修改多处
- 容易不一致

**解决方案**: 统一使用 `getNpcRole` 函数，删除内联实现
