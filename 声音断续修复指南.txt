========================================
🔊 声音断断续续 - 修复方案 🔊
========================================

症状：有声音，但卡卡的、断断续续、像接触不良

========================================
✅ 最可能有效的解决方法（按顺序尝试）
========================================

【方法 1️⃣：禁用音频增强】⭐⭐⭐⭐⭐
-----------------------------
这是最常见的原因！

1. 右键点击右下角喇叭图标
2. 选择"声音设置"或"声音"
3. 点击"更多声音设置"（或"声音控制面板"）
4. 在"播放"选项卡中：
   - 双击你的扬声器设备（有绿色勾的）
5. 点击"增强"选项卡
6. ✅ 勾选"禁用所有增强功能"
7. 点击"应用" → "确定"
8. 重启浏览器测试

【方法 2️⃣：调整音频格式】⭐⭐⭐⭐⭐
-----------------------------
采样率设置不当会导致破音

1. 右键点击右下角喇叭图标
2. 选择"声音设置"
3. 点击"更多声音设置"
4. 在"播放"选项卡中：
   - 双击你的扬声器设备
5. 点击"高级"选项卡
6. 在"默认格式"下拉菜单中：
   选择：**16 位，44100 Hz (CD 音质)**
   或：**24 位，48000 Hz (录音室音质)**
7. 点击"测试"按钮试听
8. 点击"应用" → "确定"

【方法 3️⃣：关闭独占模式】⭐⭐⭐⭐
-----------------------------
1. 同上打开扬声器属性
2. 点击"高级"选项卡
3. 在"独占模式"部分：
   ✅ 取消勾选"允许应用程序独占控制该设备"
   ✅ 取消勾选"给予独占模式应用程序优先"
4. 点击"应用" → "确定"

【方法 4️⃣：重启音频服务】⭐⭐⭐⭐
-----------------------------
1. 按 Win + R
2. 输入：services.msc
3. 找到以下服务：
   - Windows Audio
   - Windows Audio Endpoint Builder
4. 分别右键 → 重新启动
5. 重启电脑

【方法 5️⃣：更新/回滚音频驱动】⭐⭐⭐⭐
-----------------------------
更新驱动：
1. 右键开始菜单 → 设备管理器
2. 展开"声音、视频和游戏控制器"
3. 右键 Realtek Audio → 更新驱动程序
4. 选择"自动搜索驱动程序"

回滚驱动（如果更新后出现问题）：
1. 同上找到 Realtek Audio
2. 右键 → 属性 → 驱动程序选项卡
3. 点击"回退驱动程序"（如果有此按钮）

【方法 6️⃣：关闭硬件加速】⭐⭐⭐
-----------------------------
浏览器硬件加速会导致音频问题

在 Edge 浏览器中：
1. 点击右上角 ... → 设置
2. 左侧选择"系统和性能"
3. 关闭"使用硬件加速 (如可用)"
4. 重启浏览器

在 Chrome 浏览器中：
1. 点击右上角 ... → 设置
2. 左侧选择"系统"
3. 关闭"使用硬件加速模式"
4. 重启浏览器

========================================
🎯 针对 B 站的特殊设置
========================================

【B 站播放器设置】
-----------------------------
1. 打开 B 站任意视频
2. 右键点击视频画面
3. 选择"统计信息"
4. 查看是否有音频错误
5. 尝试切换播放器版本（HTML5/Flash）

【清除浏览器缓存】
-----------------------------
1. 按 Ctrl + Shift + Delete
2. 选择"缓存的图片和文件"
3. 点击"立即清除"
4. 重启浏览器

========================================
💻 高性能电脑特别注意
========================================

如果你的 CPU 占用率很高：

1. 按 Ctrl + Shift + Esc 打开任务管理器
2. 查看 CPU 占用率
3. 如果超过 80%，关闭一些后台程序
4. 特别是杀毒软件、同步软件等

========================================
🔧 高级修复（如果以上都无效）
========================================

【完全卸载音频驱动】
1. 设备管理器 → 声音、视频和游戏控制器
2. 右键 Realtek Audio → 卸载设备
3. ✅ 勾选"尝试删除此设备的驱动程序"
4. 点击"卸载"
5. 重启电脑（Windows 会自动重新安装）

【检查系统文件】
1. 按 Win + X，选择"Windows PowerShell(管理员)"
2. 输入：sfc /scannow
3. 等待扫描完成
4. 重启电脑

========================================
📊 最可能的原因排序
========================================

1. 音频增强功能未禁用 (60%)
2. 采样率设置不当 (20%)
3. 驱动问题 (10%)
4. 硬件加速冲突 (5%)
5. 其他 (5%)

========================================
💡 建议先尝试：
========================================

方法 1（禁用增强）+ 方法 2（调整格式）
这两个方法能解决 80% 的断断续续问题！
