========================================
🎓 夏威夷游戏 - 英语学习功能增强方案 🎓
========================================

现状分析：
❌ 当前问题：英语学习功能不够突出
❌ 用户感受：更像恋爱游戏，不像学英语的游戏
❌ 缺少功能：语法纠错、词汇学习、进度追踪

========================================
✅ 建议新增的英语学习功能
========================================

【功能 1：语法纠错系统】⭐⭐⭐⭐⭐
-----------------------------
功能描述：
- 玩家输入英语后，AI 自动检查语法错误
- 即时反馈：指出错误类型（时态、主谓一致等）
- 提供正确版本和中文解释
- 记录错误历史，避免再犯

实现方式：
1. 玩家发送消息后，先调用语法检查 API
2. 如果有错误，显示纠错提示卡片
3. 玩家可以重新发送正确的句子
4. 语法正确获得 XP 奖励

UI 设计：
┌─────────────────────────────────┐
│ 📝 语法小贴士                    │
├─────────────────────────────────┤
│ 你的句子：I goes to beach       │
│ ❌ 错误类型：主谓不一致          │
│ 💡 解释：第三人称单数用 goes     │
│ ✅ 正确版本：I go to beach       │
│                                 │
│ [重新发送] [跳过]               │
└─────────────────────────────────┘

【功能 2：词汇学习卡片】⭐⭐⭐⭐⭐
-----------------------------
功能描述：
- 从 NPC 对话中自动提取生词
- 显示单词卡片：音标、词性、意思、例句
- 生词本功能，随时复习
- 单词记忆游戏

实现方式：
1. NPC 说完后，AI 提取 3-5 个生词
2. 显示"今日新词"卡片
3. 点击单词查看详细解释
4. 加入生词本，后续复习

UI 设计：
┌─────────────────────────────────┐
│ 📚 今日新词 (3)                  │
├─────────────────────────────────┤
│ 🌺 traditional [trəˈdɪʃənl]      │
│    adj. 传统的                   │
│    Example: traditional culture  │
│                                 │
│ 🌺 underground [ˌʌndəˈɡraʊnd]    │
│    adj. 地下的                   │
│    Example: underground oven     │
│                                 │
│ [加入生词本] [下次复习]          │
└─────────────────────────────────┘

【功能 3：学习进度追踪】⭐⭐⭐⭐
-----------------------------
功能描述：
- 统计对话次数、学习时长
- 词汇量增长曲线
- 语法正确率趋势
- 成就系统

显示内容：
📊 学习报告
- 对话次数：50 次
- 新学单词：120 个
- 语法正确率：85%
- 连续学习：7 天
- 经验值：1250 XP

成就徽章：
🏅 对话达人（100 次对话）
🏅 词汇大师（500 单词）
🏅 语法专家（90% 正确率）
🏅 坚持不懈（30 天打卡）

【功能 4：发音评分】⭐⭐⭐⭐
-----------------------------
功能描述：
- 语音输入后，AI 评分发音
- 指出发音问题（元音、辅音、语调）
- 提供改进建议
- 录音对比功能

实现方式：
1. 集成 Web Speech API 或第三方语音 API
2. 分析发音准确度
3. 返回评分和反馈
4. 可以多次尝试

UI 设计：
┌─────────────────────────────────┐
│ 🎤 发音评分                      │
├─────────────────────────────────┤
│ 你说的：Aloha! How are you?     │
│                                 │
│ 准确度：★★★★☆ (88 分)          │
│                                 │
│ 💡 建议：                        │
│ - "Aloha"的"o"发音再饱满一些    │
│ - 语调可以更自然                │
│                                 │
│ [再试一次] [继续]               │
└─────────────────────────────────┘

【功能 5：每日挑战】⭐⭐⭐⭐
-----------------------------
功能描述：
- 每天一个主题挑战
- 完成挑战获得奖励
- 难度递进
- 排行榜

挑战类型：
📅 周一：机场对话（完成海关对话）
📅 周二：餐厅点餐（正确点餐 3 次）
📅 周三：购物砍价（成功砍价）
📅 周四：问路导航（找到目的地）
📅 周五：约会聊天（和 NPC 聊 10 轮）
📅 周末：综合挑战（卢奥晚宴）

【功能 6：学习模式切换】⭐⭐⭐
-----------------------------
功能描述：
- 简单模式：有中文翻译 + 语法提示
- 普通模式：只有中文翻译
- 困难模式：纯英语，无提示
- 专家模式：快速对话，限时回答

【功能 7：复习系统】⭐⭐⭐⭐
-----------------------------
功能描述：
- 根据艾宾浩斯遗忘曲线
- 自动安排复习时间
- 生词复习游戏
- 语法点复习

========================================
🎯 优先级排序（建议实现顺序）
========================================

第一阶段（1-2 周）：
1. ✅ 语法纠错系统（最重要！）
2. ✅ 词汇学习卡片
3. ✅ 学习进度追踪

第二阶段（2-4 周）：
4. ✅ 每日挑战
5. ✅ 复习系统
6. ✅ 学习模式切换

第三阶段（1-2 月）：
7. ✅ 发音评分（需要第三方 API）

========================================
💡 快速实现方案（最小可行产品）
========================================

如果时间有限，先实现这 3 个核心功能：

【1. 语法纠错（简化版）】
- 玩家发送后，AI 自动检查
- 有错误就显示提示卡片
- 记录到错题本

【2. 生词卡片（简化版）】
- 每段对话提取 3 个生词
- 显示简单解释
- 加入生词本

【3. 学习报告（简化版）】
- 统计对话次数
- 统计生词数量
- 显示学习天数

📊 预期效果
========================================

✅ 有成就感（成就系统）
✅ 愿意持续学习（每日挑战）

市场差异化：
✅ 比多邻国更有剧情
✅ 比传统背单词更有趣
✅ 比纯游戏更有教育价值
