🌺 夏威夷游戏 - 启动说明 🌺
========================================

✅ 服务器已启动成功！

访问地址：
http://127.0.0.1:8000

或者：
http://localhost:8000

========================================
📝 如何开始游戏：
========================================

1. 在浏览器中打开上面的地址
2. 输入你的名字
3. 点击"开始冒险"
4. 享受夏威夷之旅！

========================================
⚠️ 注意事项：
========================================

- 音乐文件是可选的，没有音乐也可以正常游戏
- 如果看到音乐文件加载失败的错误，可以忽略
- 游戏核心功能完全正常

========================================
🎮 游戏操作：
========================================

- 用英语回复 NPC 的对话
- 点击 💡 Help 获取回答建议
- 使用底部功能栏查看地点、邂逅等信息
- 按回车键发送消息

========================================
🛑 如何停止服务器：
========================================

在运行服务器的终端按 Ctrl+C 即可停止
